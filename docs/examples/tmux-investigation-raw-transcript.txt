⏺ 🦆 鸭鸭 (Duck) 是一个 Claude Code 沙盒助手。

  核心概念：
  - 定位: 让不懂技术的人也能用 Claude Code——只需打字，不需要会 bash 命令
  - 沙盒隔离: 运行在 /tmp/duck-pulse-sandbox，不会污染真实的 ~/.claude 配置
  - 自检机制: 用颜色区分 SANDBOX（绿）vs PRODUCTION（红），防止误操作生产环境
  - 徽章系统: 在 Claude Code 状态栏显示
  pulse-hit（命中前人坑）、pulse-sync（同步状态）、pulse-pipe（处理管道）等信息
  - flag 文件: 徽章读取这些文件的状态来显示不同的颜色/图标
